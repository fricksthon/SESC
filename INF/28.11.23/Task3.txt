Задача 3
    Требуется определить вес минимального остовного дерева для
    неориентированного взвешенного связного графа.

Input
    В первой строке входных данных находятся числа N и M, где N – количество
    вершин в графе, а M – количество рёбер. В каждой из последующих M строк
    записано по тройке чисел A, B, C, где A и B – номера вершин, соединённых
    ребром, а C – вес ребра.
Output
    Вывести одно число – искомый вес.

Пример:
    input:
        33
        121
        232
        313
    output:
        3
